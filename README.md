# student-performance-analysis
This project analyzes student exam scores using Python, Pandas, Matplotlib, and Seaborn.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Kaggle Notebook
## Key Analysis
- Gender-wise performance
- Test preparation impact
- Parental education effect
- Subject score distribution
- Correlation between scores
## Insights
Students who completed test preparation scored better on average.
Reading and writing scores are strongly related.
